Variable Fonts Workshop v1

Variable fonts offer graphic designers new ways to create dynamic, complex, and consistent typographic systems. While digital typefaces traditionally require multiple font files to cover a range of weights and styles, variable fonts instead store glyphs as a range of possible forms with infinite variations. In this four-week workshop, students learn how to utilize and design variable fonts for a wide variety of use cases including typographic identity systems, interpolated animations, and context-aware adjustments like optical sizing. Projects include designing graphics with variable fonts, modifying open source typefaces to introduce variable functionality, and conceiving of original variable font designs. Course instruction includes Glyphs for designing typefaces and basic HTML/CSS for animating variable fonts; no background in type design or coding is required.
